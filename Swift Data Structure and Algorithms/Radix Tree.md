# 레이딕스 트리 (`Radix tree`)

> 트라이 트리의 압축 버전
>
> * 각각의 모서리에 하나 이상의 문자 또는 단어 전부를 담을 수 있음
> * 메모리 소모와 트리를 저장할 공간을 크게 줄일 수 있음

* 좀 더 적은 수의 노드를 사용함
* 각각의 키에 좀 더 많은 정보를 담고, 모서리의 수를 줄임
* 트라이 트리를 만드는 것보다 조금 더 어려움
